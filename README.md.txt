# Global Task Manager🚀

TaskFlow is a simple full-stack Task Manager web application built using **Flask (backend)** and **HTML/CSS/JavaScript (frontend)**.  
It allows users to create, update, delete, search, and filter tasks with a smooth UI experience.

---

## Features

- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Search tasks instantly
- Filter: All / Active / Completed
- Task completion statistics
- REST API backend with unit tests

---

## Tech Stack

- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Storage: JSON file (tasks.json)
- Testing: unittest
